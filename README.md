# Expense Tracker App

An expense Tracker App built using MongoDB, Express, React and Node.

## Install Dependencies

### `npm install`

## Install React Dependencies

### `cd client`

### `npm install`

### `cd ..`

## Run App

### `npm run dev`

### Run front end (react) only

### `npm run client`

## Build

### `cd client && npm run build`

### Run production

if NODE_ENV=production

### `npm start`
